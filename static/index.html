About PII Masking

PII (Personally Identifiable Information) masking replaces sensitive data with placeholders:

Before: Hello, my name is John Doe. Email: john@example.com
After: Hello, my name is [full_name]. Email: [email]

This helps protect user privacy while keeping emails readable for analysis.
